## 📌 Problem

The Ganga River — lifeline for millions — is increasingly polluted due to:

- Untreated sewage  
- Industrial discharge  
- Agricultural runoff  

Current monitoring systems are slow and lack real-time insights, making rapid response difficult.

---

## 💡 Our Solution

An **interactive MERN stack web app** that:

✔ Displays last **10 days water quality trends**  
✔ Provides **3-day pollution forecast**  
✔ Generates **real-time alerts**  
✔ Visualizes data using **maps & charts**  
✔ Shows a simple **Ganga Health Index**

---

## ✨ Key Features

🗺 Interactive map with monitored locations  
📊 Historical + forecast data visualization  
🚨 Smart pollution alerts  
❤️ River Health Score for easy understanding  
🔮 Short-term water quality prediction
